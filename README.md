## Deployment

To deploy this project run

```bash
  npm run build
```

and then deploy with vercel

## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`DATABASE_URL`

`NEXTAUTH_SECRET`

`NEXTAUTH_JWT_SECRET`
